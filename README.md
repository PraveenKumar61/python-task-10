# Task 10 - Object Oriented Programming (OOP)

## Objective
Create a simple OOP-based Python program to simulate basic bank operations using classes and inheritance.

## Tools Used
- Python
- VS Code

## Features
- BankAccount base class
- SavingsAccount child class
- Deposit money
- Withdraw money
- Display balance
- Demonstrates encapsulation, inheritance, and method overriding

## Files
- bank_account.py → Main Python program
- screenshots/ → Output screenshot (optional)

## How to Run
1. Open terminal
2. Run command:
   python bank_account.py

## Output
Program allows creating bank accounts, depositing, withdrawing, and checking balance.
